# Cuevora Store Listing Draft

## Short Description

Premium offline teleprompter for creators, speakers, educators and presenters.

## Full Description

Cuevora is a polished teleprompter built for creators, speakers, journalists, educators, YouTubers, podcasters and professionals who need reliable prompting without subscription friction.

Write scripts, organise your library, adjust your reading experience and prompt smoothly from your Android device. Cuevora is offline-first and guest-friendly, so you can start immediately without creating an account.

Key features:

- Smooth teleprompter playback with adjustable speed
- Mirror mode for reflective teleprompter rigs
- Focus line, countdown and progress indicator
- Font size, line spacing and theme controls
- Gesture controls and Bluetooth keyboard shortcuts
- Camera overlay and dedicated record mode
- Script editor with auto-save, word count and read-time estimates
- Tags, search, sorting and compact or detailed library views
- Import and export text
- Local JSON backup and restore
- Guest mode with optional Firebase sign-in readiness
- Privacy-first design with no ads and no analytics by default

Cuevora keeps core workflows on-device. Your scripts are stored locally unless future sync is intentionally enabled and clearly disclosed.

## Target Audience

- Video creators and YouTubers
- Podcasters
- Teachers and educators
- Public speakers
- Journalists and presenters
- Remote teams recording talks, lessons or updates
- Professionals preparing polished video messages

## Privacy-Friendly Positioning

Cuevora works without an account. Scripts are stored locally by default. The app does not collect analytics by default, does not show ads and does not upload script content for the first release.
